# Games+ 

> Clone do Apple TV+ com tema voltado para jogos — primeiro projeto em SwiftUI.

## Sobre o Projeto

**Games+** é um clone do [Apple TV+](https://www.apple.com/apple-tv-plus/) desenvolvido como primeiro contato com o SwiftUI. O projeto foi construído em parceria com [@TamiresMendesS](https://github.com/TamiresMendesS) e tem como objetivo recriar a interface do Apple TV+, adaptando o tema de séries e filmes para o universo dos jogos.

O app é 100% construído em SwiftUI e foca exclusivamente na camada visual, sem funcionalidades de navegação ou consumo de dados, servindo como estudo de layouts, componentes e estilização com SwiftUI.

## Funcionalidades

- Interface inspirada no Apple TV+
- Tema visual voltado para jogos
- Telas desenvolvidas inteiramente com SwiftUI

## Como Rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/LosadaT/GamesPlus.git
   ```
2. Abra o arquivo `.xcodeproj` ou `.xcworkspace` no Xcode.
3. Selecione um simulador iOS e pressione **Run** (`⌘ + R`).

> Requisito: Xcode 14+ e iOS 16+
